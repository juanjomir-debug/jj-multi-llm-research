# iOS Assets

Place these two files here before running `npx @capacitor/assets generate --ios`:

- `icon-1024.png` — App icon, 1024x1024px, PNG, **no transparency** (Apple rejects icons with alpha channel)
- `splash-2732x2732.png` — Splash screen, 2732x2732px, PNG, background color #1c1c1c with ReliableAI logo centered

These master files are used to auto-generate all required sizes for the App Store.
